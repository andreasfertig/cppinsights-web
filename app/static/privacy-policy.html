C++ Insights Privacy Policy

Your Data

Your data always remains in your control.

The source code you typed in the editor window is send to the C++ Insights server. At the server your source code is temporarily stored just to compile it, after that the source code on the server is deleted and the result is sent back to your browser.

Web logs

C++ Insights keeps web logs to be able to track down issues with the website. These logs are kept for a short period, after which they are permanently deleted.

Cookies & Local Storage

C++ Insights uses small pieces of information stored on your computer: Cookies and Local Storage. These help to restore your source code the next time you visit C++ Insights. It also keeps the C++ Standard you have selected. This information is stored on your computer. Only during the translation it is temporarily transferred to the C++ Insights server.

They are only used with the user's permission. See the Cookie Policy for more information.

Short Links

A user can decide to create a short link using the C++ Insights web-interface. The code as well as a time stamp and all compiler/insights options are stored in a database together with the short link id. You have no guarantees, that the links will be kept forever.
