C++ Insights Cookie Policy

Browsers support the storing and sending back of small text files called "Cookies". These cookie files can be used (amongst other things) to help to track the usage of a website. See the Mozilla description of cookies for more details.

C++ Insights uses a cookie to store current state, such as:

  • Current consent status

C++ Insights uses a Local Storage to store current state, such as:

  • Your current code
  • The selected C++ Standard

This information is used to provide a better user experience, it is used at no point to track any user identifying information. For general privacy information see Privacy Policy

Note that the consent status cookie is set even if you decline cookies. Otherwise, the website would ask you all the time whether to accept cookies. In case, you do not click decline or accept no cookies at all are stored.
